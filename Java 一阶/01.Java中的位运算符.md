## Java中的位运算符
|  符号  |  作用  |  例子 |
|  :-:   | :-:    | :-:   |
| &   | 求与 | 1&1=0 1&0=0 (对于某一位的运算) |
| 丨   | 求或 | 1丨0=1 0丨0=0 (对于某一位的运算) |
| ^   | 求异或 | 1^1=0 0^1=1 (对于某一位的运算) |
| ~   | 求飞 | ~1=0 ~0=1 (对于某一位的运算) |
| <<   | 左移 | 1<<1=2 (对于int的运算) |
| >>   | 右移 | 4>>1=2 (对于int的运算) |
| <<<   | 无符号右移 | 4>>>1=2 (对于int的运算) |
