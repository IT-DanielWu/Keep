## 工作常用命令
| 命令 | 描述 |
| ---- | ---- | 
| git init | 仓库的初始化 |
| git pull | 拉取远程仓库分支代码 |
| git checkout | 切换当前分支 |
| git merge | 合并分支 |
| git add | 将文件放进暂存区 |
| git commit | 提交代码 |
| git push | 推送分支代码到远程仓库 |
| git status | 查看当前仓库的状态 |
| git diff | 查看仓库与上次修改的内容 |
| git clone | 克隆代码 |
| git bransh | 查看当前分支 |
